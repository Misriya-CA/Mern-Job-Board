MERN Job Board – Full Stack Project

A simple and responsive Job Board platform built using the MERN stack.

Features:
-View job listings
-Post a new job
-Search jobs
-Filter by category
-Responsive design
-Full MERN setup

Tech Stack:
-React,Javascript,CSS(frontend)
-Node.js+Express(backend)
-MongoDB(database)

Tools:
-Postman
-Git & GitHub


Project Structure:
root
│
├── job-backend
│   ├── server.js
│   ├── routes/
│   ├── models/
│   ├── config/
│   └── .env
│
└── job-frontend
    ├── src/
    ├── components/
    ├── App.jsx
    ├── index.css
    └── vite.config.js


How to run:
Clone the Repository:git clone https://github.com/Misriya-CA/Mern-Job-Board.git

Frontend:
-cd job-frontend
-npm install
-npm run dev
-Frontend runs at:http://localhost:5173/


Backend:
-cd job-backend
-npm install
-node server.js
-Backend runs at: **http://localhost:5000**

API Endpoints:

GET/jobs:
-Fetch all job postings.

POST/jobs:
-Submit a new job.
